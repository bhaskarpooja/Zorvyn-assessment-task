# 💰 Finance Dashboard – Full Stack App

## 📌 Overview

Full stack finance dashboard with role-based access control, financial record management, and summary analytics.

---

## 🚀 Features

* User roles: Viewer, Analyst, Admin
* CRUD for financial records
* Filtering (date, category, type)
* Dashboard: income, expenses, balance, trends
* Validation & error handling

---

## 🏗️ Tech Stack

* Backend: Java, Spring Boot
* Frontend: React
* DB: PostgreSQL (prod), H2 (dev)

---

## ⚙️ Structure

```
backend/
frontend/
```

---

## 🧪 Run Locally

```
# backend
cd backend
mvn spring-boot:run

# frontend
cd frontend
npm install
npm start
```
